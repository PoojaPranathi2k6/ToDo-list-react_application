# To-Do List Application

## Overview

This is a React.js To-Do List Application developed as part of the Codec Technologies Internship. It helps users manage daily tasks with an easy-to-use interface.

## Features

* Add new tasks
* Delete tasks
* Update task status (Pending → In Progress → Completed)
* Filter tasks by status
* Sort tasks alphabetically
* Store tasks using Local Storage
* Display task statistics
* Clear all tasks with confirmation
* Modern lavender-themed user interface

## Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Local Storage

## Installation

1. Clone the repository.
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
